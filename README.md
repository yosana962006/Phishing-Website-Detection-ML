# Phishing Website Detection using Machine Learning

## Overview

This project detects phishing websites using Machine Learning algorithms. Phishing attacks are one of the most common cyber threats where attackers create fake websites to steal sensitive information such as passwords, banking credentials, and personal data.

The project uses a supervised Machine Learning approach to classify websites as:

* Legitimate Website
* Phishing Website

## Dataset

The dataset contains multiple website-related security features such as:

* URL Length
* HTTPS Token
* SSL Final State
* Domain Age
* DNS Record
* Google Index
* Web Traffic
* Request URL
* URL of Anchor
* Prefix/Suffix

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Machine Learning Model

Random Forest Classifier was used for training the model.

### Model Performance

* Accuracy: 97.46%
* Precision: 97%
* Recall: 97%
* F1-Score: 97%
